# 🗄️ Northwind MVC - Database First (EF Core)

An ASP.NET Core MVC application built using **Entity Framework Core Database-First approach** based on the classic **Northwind database**.

This project demonstrates how to scaffold an existing database into models and DbContext, and implement full CRUD operations using MVC architecture.

---

# 🚀 Features

## 🧱 Database First Approach
- Reverse engineering Northwind database using EF Core Scaffolding
- Auto-generated DbContext and entity models
- Mapping relational database tables to C# classes

---

## 📦 Product Management (CRUD)
- Create new products
- Edit existing products
- Delete products
- View product details
- List all products with pagination-ready structure

---

## 🔗 Relationships Handling
- Products ↔ Categories relationship
- Proper foreign key navigation properties
- Include related data using EF Core eager loading

---

## 🎨 UI Layer
- Built using Razor Views
- Clean MVC separation (Model - View - Controller)
- Bootstrap 5 responsive UI

---
